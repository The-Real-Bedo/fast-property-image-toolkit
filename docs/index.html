What's in the toolkit
Every feature you need

Production-ready out of the box. Read the full docs on GitHub →

WebP conversion

30–50% smaller than JPEG. Sharp handles encoding with optimal settings.

3 size variants

Thumbnail 400px · Mobile 800px · Desktop 1920px — all in parallel.

Blur placeholder

~400 byte Base64 WebP. Instant visual feedback — zero extra request.

Responsive srcset

Browser picks the right resolution automatically per device.

Lazy loading

IntersectionObserver — off-screen images are never downloaded.

EXIF auto-rotate

Sideways phone photos fixed before processing — automatically.

Input validation

MIME whitelist + 50 MB cap. Bad files rejected before any bytes are read.

Full TypeScript

Every type exported. Frontend interfaces mirror backend JSDoc exactly.

Quick start
Up and running in 3 commands

Full documentation and installation guide on GitHub →

Terminal
# Clone and install
git clone https://github.com/The-Real-Bedo/fast-property-image-toolkit.git
cd fast-property-image-toolkit && npm install

cp packages/backend/.env.example packages/backend/.env
npm run dev:backend
# → http://localhost:4000/api/images/process ✅
React
import { PropertyImage } from './components/PropertyImage';

function PropertyCard({ property }) {
  return <PropertyImage data={property.image} alt={property.title} aspectRatio="3/2" />;
}
