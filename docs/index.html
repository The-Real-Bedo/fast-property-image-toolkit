What's in the toolkit
Everything you need

Production-ready out of the box. Full docs on GitHub →

WebP conversion

30–50% smaller than JPEG at equivalent quality.

3 size variants

Thumbnail 400px · Mobile 800px · Desktop 1920px in parallel.

Blur placeholder

~400 byte Base64 WebP shown instantly — zero extra request.

Responsive srcset

Browser picks the right resolution per device automatically.

Lazy loading

IntersectionObserver — off-screen images are never downloaded.

EXIF auto-rotate

Sideways phone photos corrected automatically before processing.

Input validation

MIME whitelist + 50 MB cap — bad files rejected early.

Full TypeScript

Every type exported. Frontend interfaces mirror backend types.

Get started
3 commands to run it
Terminal
# Clone and install
git clone https://github.com/The-Real-Bedo/fast-property-image-toolkit.git
cd fast-property-image-toolkit && npm install
cp packages/backend/.env.example packages/backend/.env
npm run dev:backend  # → localhost:4000 ✅
React
import { PropertyImage } from './components/PropertyImage';
<PropertyImage data={property.image} alt={property.title} aspectRatio="3/2" />
